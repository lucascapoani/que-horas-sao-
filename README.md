# Que horas são?

 <h2> :computer: Projeto </h2>
 <p> Projeto criado para declarar ao usuário a hora atual. Além de mostrar a hora atual, o sistema mostra uma imagem referente ao momento do dia: manhã, tarde ou noite.</p>
<hr>
